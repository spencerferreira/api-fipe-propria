Política de Privacidade — Super Fipe

Última atualização: 20/01/2025

Esta Política de Privacidade descreve como o aplicativo Super Fipe (“App”) trata informações quando você o utiliza. O App é disponibilizado por Eng. Spencer Ferreira (“Desenvolvedor”).

1. Escopo e aceitação

Ao usar o App, você concorda com esta Política de Privacidade. Se você não concordar, interrompa o uso do App.

2. Dados que o App coleta

2.1. Dados coletados diretamente pelo Desenvolvedor

O Desenvolvedor não solicita cadastro e não coleta diretamente dados pessoais como nome, e-mail, telefone ou documentos. O App não exige login para funcionar.

2.2. Dados coletados automaticamente por terceiros

O App utiliza serviços de terceiros (como a pesquisa de imagens do Google) para exibir fotos de veículos. Esses parceiros podem coletar dados para exibição de conteúdo, anúncios, medição, prevenção a fraude e melhoria de desempenho.

  • Endereço IP e identificadores de dispositivo.
  • Interações com o produto e informações de diagnóstico (falhas e desempenho).

3. Como os dados são usados

Os dados coletados por serviços de terceiros são usados para exibir o conteúdo solicitado (imagens), medir desempenho técnico, segurança e melhoria de estabilidade do App.

4. Compartilhamento de dados

O Desenvolvedor não vende dados pessoais. O compartilhamento ocorre automaticamente com provedores de serviços para as finalidades técnicas descritas.

5. Permissões e acesso do dispositivo

  • INTERNET: Usada para realizar as consultas à Tabela Fipe e carregar imagens de veículos via Google.

6. Conteúdo e imagens obtidas na internet

O App exibe imagens obtidas via links públicos apenas para fins referenciais. O conteúdo permanece sob responsabilidade de seus respectivos provedores originais.

7. Segurança da informação

O App utiliza criptografia em trânsito (HTTPS/TLS) para transmissão de dados, conforme práticas modernas de segurança.

8. Retenção e exclusão de dados

O Desenvolvedor não mantém banco de dados próprio com informações pessoais. Dados coletados por terceiros seguem as políticas de retenção de seus respectivos provedores.

9. Direitos do usuário

Você pode limitar anúncios personalizados nas configurações do seu Android. Como não coletamos dados identificáveis diretamente, não há perfis de usuário para gerenciar.

10. Crianças e público-alvo

O App não é direcionado especificamente a crianças. O conteúdo é destinado ao público interessado em informações automotivas.
